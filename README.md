# Summer-internship-assignment


This repository contains solutions for all tasks given in the Summer Internship Assignment.

## Assignment Tasks

### Task 1 — List & Dictionary Operations
- Remove duplicates from a list
- Sort the list
- Find maximum, minimum, average
- Count frequency of elements

### Task 2 — File Handling
- Read log file
- Count log types
- Save results in JSON
- Find most frequent log type

### Task 3 — Functions & Error Handling
- Implement safe_divide(a, b)
- Handle division by zero
- Handle invalid inputs
- Add proper docstrings

### Task 4 — Debugging
- Fix buggy list removal code
- Explain the issue
- Provide optimized solution
- Analyze time complexity

### Task 5 — Data Cleaning
- Load CSV dataset using pandas
- Handle missing values
- Remove duplicates
- Encode categorical columns
- Save cleaned dataset

### Task 6 — Data Visualization
- Create visualizations:
  - Histogram
  - Scatter Plot
  - Heatmap
  - Bar Chart
  - Boxplot
- Save plots as image files

### Task 7 — Insights Report
- Analyze trends
- Mention dataset issues
- Summarize conclusions

### Task 8 — Web Scraper
- Scrape Twitter/X posts
- Extract:
  - Tweet text
  - Username
  - Tweet link
  - Media/video links
  - Timestamp
  - Engagement statistics
- Save data in CSV and JSON

### Task 9 — Context Based Scraping
- Accept multiple keywords
- Merge search results
- Remove duplicates
- Save categorized videos

### Task 10 — Data Summary & Analysis
- Analyze collected data
- Generate summary report
- Create visualizations
